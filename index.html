Task and Methods

The premise of this task is to read image using python opencv, and manipulate the image array to make a grayscale version and a blurred version of that image.

For grayscale image, my algorithms is to take the average of the RGB value and replace each pixel with a single channel.

For blurred image, my algorithms is to assign each pixel to their closest topleft pixel whose x,y coordinates are divisble by 10.
